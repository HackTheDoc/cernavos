# Cernavos

Tiny rts game project from scratch.

# Table of Content

- General Info
- Technologies
- Installation

# General Info

The game is a real time strategy game.
It features a unique unit creation system and a PvP and PvAI modes.

> Both PvP and PvAI are currently under development.

# Technologies

- SDL2
